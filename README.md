# IoT In Godot In Seven Hours

> The workshops in here aim to be pratice in the context of formation center around Belgium in one day (6-8 hours.)

**Concept:** Godot is an interpreted language, it means that players can add their code in the game.
Using existing simulation of component used in Iot industry, try to create a Robot for the fun of learning.
